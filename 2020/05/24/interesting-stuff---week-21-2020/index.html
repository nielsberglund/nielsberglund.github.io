Interesting Stuff - Week 21, 2020

May 24, 2020 in ROUNDUP
distributed computing sql server machine learning azure data studio kafka ksqldb azure flink
2 min read

Throughout the week, I read a lot of blog-posts, articles, and so forth, that has to do with things that interest me:

  • data science
  • data in general
  • distributed computing
  • SQL Server
  • transactions (both db as well as non db)
  • and other “stuff”

This blog-post is the “roundup” of the things that have been most interesting to me, for the week just ending.

Distributed Computing

  • SLOG: serializable, low-latency, geo-replicated transactions. In this blog post Murat dissects a white paper about how one can in a distributed system avoid the tradeoff of giving up one of serializability, low latency writes, or high transactional throughput. Murat does a great job dissecting a complex topic and make it understandable.

SQL Server

  • Machine Learning extension (preview) for Azure Data Studio. The May 2020 release of Azure Data Studio was announced a week or so ago, and in that release was an extension for Machine Learning. The link here is to the documentation of the extension. It looks interesting, and I’ll try to take it for a “spin” as soon as I get time.

Streaming

  • How GoDaddy uses Flink to run real time streaming pipelines. This post, as the title says, looks at data pipelines at GoDaddy, and how they use Apache Flink in conjunction with other tools to create and run those pipelines. What I found especially interesting in the post was the description of how they deployed a new pipeline into production. That is something I will copy and try to implement at Derivco.

WIND (What Is Niels Doing)

On Tuesday, (May 26), I am co-hosting the first The Azure Sketches, Architect Series webinar together with Gordon Beeming and Lawrance Reddy. This series will use a combination of traditional drawing & sketching with actual solution building to help reinforce the thought process in building Azure solutions and all of the dependencies in between. If you are interested, here is the link for registration.

Thursday, May 28, I do another first, the first-ever Derivco Webinar. Charl Lamprecht, who is Derivco’s Kafka Guru, and I are delivering a webinar about Kafka and ksqlDB: Conquer Your Data with Kafka and ksqlDB. If you are interested, you can register here.

I hope to see you at either of these events!

~ Finally

That’s all for this week. I hope you enjoy what I did put together. If you have ideas for what to cover, please comment on this post or ping me.
